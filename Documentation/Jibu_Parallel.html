Remarks

Parallel is a static class providing some basic parallel methods such as For, Reduce, Run and Gather. 

All methods, except some versions of Run, are implemented with Jibu's built-in scheduler that allows multiple tasks to run efficiently on the available processors/cores. It's important to understand that the built-in scheduler is not preemptive, meaning that it's not meant for executing tasks that run forever. If a task temporarily blocks, the scheduler detects it and ensures that the processor is kept busy executing another task. For more information about the scheduler, see the Task class.
