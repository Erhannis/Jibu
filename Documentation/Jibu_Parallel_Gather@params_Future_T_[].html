Collects individual results from multiple Futures.

    params Future<T>[] tasks

Visual Basic
Public static Function Gather(Of T)(
    tasks As params Future<T>[]
) As T()
Parameters

tasks
The Futures to be run concurrently
T
The type of data returned from each Future

Returns

An array of type T containing the Future results

Remarks

Parallel.Gather executes multiple Futures concurrently and returns an array containing the result of each individual Future. Results must be of the same type. 

Parallel.Gather catches any uncaught exception raised inside the Futures. Once the exception is caught Parallel.Gather cancels all running Futures that it has started and throws a Jibu.CancelException, containing the original exception. Jibu cancels all Futures but it doesn't just brutally stop them - the user defined code must detect and handle the cancellation. For more information on cancellation see Jibu.Task.Cancel and Jibu.CancelException

Exceptions
Exceptions 
Description 
Jibu.CancelException 
If one of the Futures is cancelled, Parallel.Gather cancels the remaining Futures and throws a Jibu.CancelException. 
Parallel Gather Example
Copy Code
using System;
using System.Collections.Generic;
using System.Text;
using Jibu;

namespace GatherExample
{
   // This example shows how to run Futures concurrently.
   public class GatherExample
   {
       static void Main(string[] args)
       {
           // Run two Futures concurrently and gather the results.
           int[] results = Parallel.Gather<int>
           (
            new DelegateFuture<int>(delegate
            {
                // Add all numbers from 0 to 999
                return Parallel.Reduce<int>(0, 1000, 0,  1000 / 10, 
                    delegate(int i) { return i; }, 
                    delegate(int x, int y) { return x + y; });
            }),
            new DelegateFuture<int>(delegate
            {
                // Add all numbers from 1000 to 1999
                return Parallel.Reduce<int>(1000, 2000, 0, 
                    delegate(int i) { return i; }, 
                    delegate(int x, int y) { return x + y; });
            })
           );

           Console.WriteLine("1 + 2 + 3 + ... + 999 =    " + results[0]);
           Console.WriteLine("1000 + 1001 + ... + 1999 = " + results[1]);
           Console.WriteLine("nPress any key to continue ...n");
           Console.ReadKey();

       }
   }
}
