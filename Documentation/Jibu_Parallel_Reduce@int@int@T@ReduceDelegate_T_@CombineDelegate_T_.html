Parameters

start
First index, which is incremented by one until it equals end.
end
Last index, which is not included in the loop. The loop runs as long as start is less than end.
initialValue
The initial value for the result.
loopbody
A delegate containing the work. Loopbody is executed once for each iteration.
combineMethod
A delegate used to combine the local results
T
A type that must be associative

Returns

A single value of type T

Remarks

Parallel.Reduce combines multiple associative values into a single value. The ReduceDelegate loopbody takes an integer between start and end and returns an element of type T. The CombineDelegate takes two such elements and combine them.  

Parallel.Reduce avoids using locks by dividing the problem into a binary task tree where each task updates it's own local value. 

Grain size is implicit set to two. It's often possible to increase performance by adjusting the grain size. To do so use Parallel.Reduce(int start, int end, T initialValue, int grainSize, ReduceDelegate loopbody, CombineDelegate combineMethod). Parallel. 

Reduce catches any uncaught exception raised inside the tasks. Once the exception is caught Parallel.Reduce cancels all running tasks that it has started and throws a Jibu.CancelException, containing the original exception. Jibu cancels all tasks but it doesn't just brutally stop them - the user defined code must detect and handle the cancellation. For more information on cancellation see Jibu.Task.Cancel and Jibu.CancelException

Exceptions
Exceptions 
Description 
Jibu.CancelException 
If one of the Tasks is cancelled, Parallel.Reduce cancels the remaining Tasks and throws a Jibu.CancelException. 
Parallel Reduce Example
Copy Code
using System;
using System.Collections.Generic;
using System.Text;
using Jibu;

namespace ReduceExample
{
    // This example shows how to combine multiple
    // associative values into a single value, using
    // the Reduce method.
    class ReduceExample
    {
        static void Main(string[] args)
        {
            // Adding integers from 0 to 1000000.
            // loopbody simply returns the index while combineMethod add the numbers.
            int sum1 = Parallel.Reduce<int>(0, 1000000, 0, delegate(int i) { return i; }, delegate(int x, int y) { return x + y; });
            Console.WriteLine("1 + 2 + 3 + ... + 1000000 = " + sum1);
            

            // Adding integers from -10 to 1000000.
            // The initial value is 0 and the grainSize is set to 10000.
            // loopbody simply returns the index while combineMethod add the numbers.
            int sum2 = Parallel.Reduce<int>(-10, 1000000, 0, 10000, delegate(int i) { return i; }, delegate(int x, int y) { return x + y; });
            Console.WriteLine("nn-10 + (-9) + (-8) + ... + 1000000 = " + sum2);
        }
    }
}
