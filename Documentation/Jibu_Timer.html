A timer that can be used as a standalone timer or as part of Choice

Remarks

To use the Timer as a regular timer, use the static SleepFor method, which will block for the specified period of time. In the Choice class you might want to be notified if the Timer times out. In that case you should use RelativeTimeOut instead. This method is non-blocking, but when the Timer times out after the specified period of time it will notify the Choice instance associated with the Timer.

Notes

A Timer instance is only meant to be used in a single Choice/task. If more Timers are needed create more instances.

Timer Example
Copy Code
using System;
using Jibu;

namespace TimerExample
{
    // Demonstrates the Timer class
    class TimerExample
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Sleeping for 2 seconds");

            // We simply sleep for 2 seconds
            // with this static method
            Timer.SleepFor(2000);
            Console.WriteLine("Aahhh, a little nap is always nice.");


            // We create a timer instance
            Timer timer = new Timer();

            // Timer can also be included in a Choice. 
            // We also create a Channel istance
            // to take part.
            Channel<int> chan = new Channel<int>(JibuBuffer.Fifo, 2);
            Choice c = new Choice(timer, chan.ChannelReader);

            // make the timer time out in 
            // 1 second
            timer.RelativeTimeOut(1000);


            // We should be notified by the Timer -> index 0            
            int index = c.FairSelect();
            Console.WriteLine("We were notified by {0}", index);

            // Reschedule timer, so it's no longer
            // timed out.
            timer.RelativeTimeOut(4000);

            // Write to the channel
            chan.Write(42);

            // We should be notified by the Channel -> index 1
            index = c.FairSelect();
            Console.WriteLine("We were notified by {0}", index);
            if(index == 1)
                Console.WriteLine("We got {0} from the channel.", chan.Read());

        }
    }
}
